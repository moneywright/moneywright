Moneywright is a desktop app that runs entirely on your computer. Download the installer for your operating system and you're ready to go.

## Desktop App (Recommended)

The desktop app provides the easiest setup experience — just download, install, and run.

<DownloadTable />

### macOS

1. Download the DMG file for your Mac (Apple Silicon or Intel)
2. Open the DMG and drag Moneywright to your Applications folder
3. Launch Moneywright from Applications
4. **First launch:** Right-click the app and select "Open" to bypass Gatekeeper (macOS security)

<Callout type="info">
**Apple Silicon vs Intel:** If you have an M1, M2, M3, or M4 Mac, download the Apple Silicon version. For older Macs (pre-2020), download the Intel version.
</Callout>

### Windows

1. Download the Windows installer (.exe)
2. Run the installer and follow the prompts
3. Launch Moneywright from the Start menu or desktop shortcut

<Callout type="info">
**Windows SmartScreen:** You may see a "Windows protected your PC" warning. Click "More info" then "Run anyway" to proceed.
</Callout>

### Linux

1. Download the DEB package
2. Install with: `sudo dpkg -i moneywright_*.deb`
3. Launch Moneywright from your applications menu

For other Linux distributions, use the [command line install](#command-line-install) below.

## Command Line Install

Install Moneywright CLI with a single command. This downloads the latest release binary and sets it up automatically.

### macOS / Linux

```bash
curl -fsSL https://moneywright.com/install.sh | bash
```

### Windows (PowerShell)

```powershell
irm moneywright.com/install.ps1 | iex
```

### What the installer does

- Detects your operating system and architecture
- Downloads the appropriate binary from GitHub releases
- Installs to `~/.moneywright` (Unix) or `%LOCALAPPDATA%\Moneywright` (Windows)
- Optionally adds `moneywright` to your PATH

### Running the CLI

After installation, start the server:

```bash
moneywright
```

Then open [http://localhost:17777](http://localhost:17777) in your browser.

### Updating

Run the install script again to upgrade to the latest version. Your data and configuration will be preserved.

## Building from Source

For developers who want to build Moneywright from source.

### Prerequisites

- [Bun](https://bun.sh) v1.1 or later
- [Node.js](https://nodejs.org) v20 or later
- [Rust](https://rustup.rs) (for desktop app only)

### Development Setup

```bash
# Clone the repository
git clone https://github.com/moneywright/moneywright.git
cd moneywright

# Install dependencies
bun install

# Generate environment variables
bun run setup

# Run database migrations
bun run db:migrate

# Start development servers
bun run dev
```

This starts both the API server (port 17777) and web dev server (port 3000).

### Building Binaries

Build a standalone binary for distribution:

```bash
# Current platform
bun run build:binary

# Specific platforms
bun run build:binary:macos         # macOS ARM64
bun run build:binary:macos-intel   # macOS Intel
bun run build:binary:linux         # Linux x64
bun run build:binary:windows       # Windows x64
```

### Building Desktop App

Build the native desktop application:

```bash
# Current platform
bun run build:desktop

# Specific platforms
bun run build:desktop:macos     # macOS (DMG)
bun run build:desktop:windows   # Windows (NSIS)
bun run build:desktop:linux     # Linux (DEB)
```
