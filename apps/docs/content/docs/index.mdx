Moneywright is an open-source, self-hostable, AI-powered personal finance helper. Upload your bank statements, credit card statements, and investment data to get intelligent expense analysis, investment insights, and personalized financial advice.

## Features

- **AI-Powered Analysis** - Get intelligent insights into your spending patterns
- **Statement Upload** - Import bank and credit card statements
- **Expense Tracking** - Automatic categorization of expenses
- **Investment Analysis** - Track your portfolio performance
- **Financial Advice** - Personalized recommendations
- **Self-Hostable** - Full control over your financial data

## Quick Start

The fastest way to get started is with the desktop app or one-line installer:

```bash
# macOS / Linux
curl -fsSL https://moneywright.com/install.sh | bash

# Windows (PowerShell)
irm moneywright.com/install.ps1 | iex
```

Then open [http://localhost:17777](http://localhost:17777) in your browser.
