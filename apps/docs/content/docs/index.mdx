Moneywright is a free, open-source personal finance app that runs on your computer. Upload your bank statements, track expenses and investments, and get AI-powered insights — all without your data leaving your machine.

## Why Moneywright?

- **100% Private** — Your financial data stays on your computer. No cloud sync, no accounts, no data harvesting.
- **Works with Any Bank** — If your bank gives you a PDF or CSV statement, Moneywright can read it.
- **AI-Powered** — Smart categorization, subscription detection, and a chat assistant that answers questions about your finances.
- **Free Forever** — Open source, no premium tiers, no subscriptions.

## Quick Start

**1. Download the desktop app**

<DownloadTable />

**2. Add your AI provider** (optional but recommended)

Moneywright uses AI to categorize transactions and power the chat assistant. You bring your own API key from OpenAI, Anthropic, Google, or run locally with Ollama.

→ [Set up AI](/docs/ai-setup)

**3. Import your first statement**

Drop a PDF or CSV bank statement into Moneywright. The AI will extract and categorize every transaction automatically.

→ [Importing statements](/docs/importing-statements)

## Alternative: Command Line Install

If you prefer the terminal, install with a single command:

```bash
# macOS / Linux
curl -fsSL https://moneywright.com/install.sh | bash

# Windows (PowerShell)
irm moneywright.com/install.ps1 | iex
```

Then run `moneywright` and open [http://localhost:17777](http://localhost:17777).
